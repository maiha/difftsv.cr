# difftsv.cr

difftsv for [Crystal](http://crystal-lang.org/).

- crystal: 0.32.1

## Installation

1. Add the dependency to your `shard.yml`:

```yaml
dependencies:
  difftsv:
    github: maiha/difftsv.cr
```

2. Run `shards install`
